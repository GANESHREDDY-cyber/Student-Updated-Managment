# KL Campus Connect - Deployment Guide

## Quick Deploy Steps:

1. **Create GitHub Repository:**
   - Repository name: `kl-campus-connect`
   - Make it public

2. **Deploy Commands:**
   ```bash
   git init
   git add .
   git commit -m "KL Campus Connect - Student Activity Platform"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/kl-campus-connect.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to Settings → Pages
   - Source: Deploy from branch
   - Branch: main
   - Save

4. **Your Live URL:**
   `https://YOUR_USERNAME.github.io/kl-campus-connect/`
